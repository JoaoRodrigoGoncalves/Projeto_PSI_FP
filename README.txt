NOTA IMPORTANTE:

Por algum motivo, o codeblocks guarda os ficheiros de dados num local diferente do ficheiro executável.
Visto isto, se o programa reportar que não existem ficheiros de dados, pode ser necessário copiar ou mover os ficheiros que estão na raíz do projeto para a pasta onde se encontra o executável.
