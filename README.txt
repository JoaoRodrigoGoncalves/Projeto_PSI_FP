NOTA IMPORTANTE:

Por algum motivo, o codeblocks guarda os ficheiros de dados num local diferente do ficheiro executável.
Visto isto, e de forma a ser mais fácil de experimentar o programa, os mesmos ficheiros de dados estão copiados em ambos os diretorios.
